# simple-movie-QA-system
极简电影问答系统

# 相关内容参考

# 使用

- 安装好需要的第三方库

```shell
pip install -r requirements.txt
```

- 配置好pyhanlp自定义词典（建议动态加载）

- 启动程序

```shell
python main.py
```
